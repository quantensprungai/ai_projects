# code/

Hier liegen **Code-Repositories** (jeweils als eigenes Git-Repo), getrennt von der reinen Doku unter `projects/`.

Beispiel:

```text
code/
  annas-archive-toolkit/   (Git-Repo)
  trading-bot/             (Git-Repo)
  hd-saas/                 (Git-Repo)
```
