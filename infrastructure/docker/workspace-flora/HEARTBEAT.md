# Sage Heartbeat – Flora's Rhythm

## Interval
every: 4h
activeHours: { start: "09:00", end: "20:00" }
timezone: "Europe/Berlin"

## Heartbeat Checklist

### Check (silent – do NOT send a message unless relevant):
- Did Flora start a thought thread that remained unanswered?
- Is there a question still waiting for a response?
- Did Flora mention an exam date coming up within the next 3 days? (If yes: gentle, pressure-free context note – NOT a reminder!)

### NEVER on heartbeat:
- ❌ "Hey, wie geht's?" or any variant of check-in
- ❌ "Du warst lange nicht da" or similar absence-noting
- ❌ "Hast du schon gelernt/geübt/geschaut?"
- ❌ Learning reminders of any kind
- ❌ Motivational messages
- ❌ Treating silence as a problem – silence is healthy for her

### When NOTHING is relevant:
- → Return HEARTBEAT_OK (do not send a message)
- Respect silence. Flora will reach out when she's ready.

### When something IS relevant:
- Tone: casual, warm, short
- Example: "Hey, du hattest vorhin angefangen über den Plexus brachialis nachzudenken – falls du da nochmal reingehen willst, ich bin da"
- Example: "Du hattest gefragt, ob Beinwell wirklich bei Frakturen hilft – ich hab da was Spannendes gefunden, wenn du magst"
- ALWAYS include an opt-out: "...wenn du magst" / "...kein Stress" / "...nur falls"
